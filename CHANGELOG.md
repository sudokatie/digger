# Changelog

All notable changes to Digger will be documented in this file.

## [0.1.1] - 2026-02-01

### Fixed
- Next Level button now works after completing a level
- Best completion times now tracked correctly (lower is better)

## [0.1.0] - 2026-02-01

### Added
- Core movement system (run, climb, hang, fall)
- Digging mechanics with hole timers
- Guard AI with A* pathfinding
- Gold collection and exit reveal
- 10 playable levels
- Level select screen
- HUD with gold counter, lives, and timer
- Pause menu with controls help
- Win/lose screens
- Keyboard controls (arrows, WASD, Q/E for dig)

### Technical
- Next.js 14 + TypeScript foundation
- Canvas-based rendering
- 170 unit tests
- ~96KB first load JS
